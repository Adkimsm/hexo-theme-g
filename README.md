# Hexo-Theme-G

A theme Form Typecho.

(已进入任意发展时期)

## 使用

将整个文件夹重命名为 `g` ，放入`站点根目录/themes/`目录。

将站点根目录内的`config.yml` 中的 `theme` 字段改为 g.

在根目录运行：

```sh
hexo clean
hexo s
```
